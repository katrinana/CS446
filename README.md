# Must Wake Up

## Usage
A mobile application of alarm clock whick help people who have trouble wake up in the morning. You must finish some task to close the alarm.

### Android

download the file:

    android-debug.apk.1
    

### Desktop

In your browser, open the file:

    /www/index.html
